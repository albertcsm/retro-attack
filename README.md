## Development

### Prerequisites

- Node.js
- Rust

### Setup

1. Clone the repository
    ```
    git clone https://github.com/albertcsm/retro-attack.git
    ```

2. Install dependencies
    ```
    npm install
    ```

3. Run the development server
    ```
    npm run start
    ```

4. Open your browser and navigate to `http://localhost:9000` to see the game.
